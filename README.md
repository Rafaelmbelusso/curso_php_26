# INFORMAÇÕES VSCODE

- Grafias: Camel Case --> AvIaO123
           Snake Case --> aviao123

- Inserir Bloco de cometários: Na linha ---> //
                               Bloco ---> /** 

- Declarar ou informar variável: $ --> **$aviao123**

- Atribuição: = ---> $aviao123 **=** 

- String: "(teste)" ---> $aviao123 = **"teste"**

- Operador de junção de partes: . ---> $aviao123 = "teste" **.** $aviao456 = "teste2"
